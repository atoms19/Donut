Donwload Donut

Donut is simply a stylesheet.
You can download the latest version here.
You can also fit it in your head tag like this :

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Wixonic/Donut@2.0/Donut.min.css" />

You can select text (if tag has .selectable class).
All code blocks of this documentation have this class.

Next, you must add a special tag in which you will put the content of the page you want to stylize :

<donut></donut>

And now, you're ready !


Use Donut

Donut does not change the construction of your web page. Just add HTML tags and the rendering will be stylized.
